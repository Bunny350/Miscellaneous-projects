# Thermometer AA battery mod
This mod allows replacement of button-cell battery of the rectangular-shaped thermometer-hygrometer units to use AA batteries for ultra-long battery life. This allows ditching alkaline cells, if you know what I mean (it's Ni-MH cells). They are all designed by me.

## Variants
There are two variants, normal and pro version[^follower-exclusive].

## Print preparation
* **The smooth build plate is recommended just to make the usability easier. Unfortunately, not all providers, vendors or manufacturers sell smooth build plates for the printers, and may only be textured. If you only have textured build plates for any printers but not smooth plates for any of them, you are stuck here unless you figure out on how to work around weak adhesive bonding on the standard version or air leak on the pro version[^follower-exclusive].**

## BOM
Both variants can use the same materials, with the exception on how they are secured.

### Both variants

* 1X Rectangular thermometer & hygrometer unit (with or without extension cable / tail)
* 1X 1S AA battery tray (with wires included)
* 1X 10mm thick VHB tape (gray) (or double-sided foam tape with 0.5 to 0.6mm thickness)

### Pro version[^follower-exclusive]-exclusive

* 1X 20x0.5mm foam tape
    * Usually comes in 5-meter length.
* 1X 5mm multi-sided tape
    * The foam tape above can be used if the multi-sided tape is not available.
* 4X M3x4x5mm heat-set insert
* 4X M3x6mm BHCS
* The following materials are opt-in:
    * 1X 10x3mm foam tape
        * Used to seal holes and cutouts.

## Installation
Both variants can follow the same procedure, with the exception of on how they are mounted.

* On the pro version, M3x4.5x5 heat-set inserts need to be inserted first. Place four heat-set inserts to the respective holes.
    * [^follower-exclusive]

* Attach the battery tray first. Use the VHB tape to secure the battery tray.
    * <img src="https://raw.githubusercontent.com/Bunny350/Miscellaneous-projects/main/Thermometer%20AA%20battery%20mod/Media/Assembly/IMG_4213.jpeg" height="400" />
    * The wires should face down on the standard version while they should face up on the pro version.
    * On the pro version, tuck the wires to the inside compartments.
