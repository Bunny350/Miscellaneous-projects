# Bunny350 Adventure
This folder contains materials for this Java world. This world is intended to be played by one player.

[Download it as archive](https://download-directory.github.io?url=https://github.com/Bunny350/Miscellaneous-projects/tree/main/Abandoned%20projects/Bunny350%20Adventure)

## Requirements
* Specific Minecraft JE version 1.12 to 1.12.2.
    * Because 1.13 has changed how commands formatted, most specifically major change to /execute, killing or merging /testfor(block(s)) by or to /execute (if entity / block), and it also cause unexpected warning messages.
        * Issues related to this will **not** be fixed and will be marked **invalid**, as this map was marked scrapped and will not be developed further.
    * If used in earlier versions, commands in the blocks will be removed, and the map will never function commands.
    
## Last-hour changes
This map has been adjusted for public process. As we want it to be reusable (no disposal after overall), this map has a factory reset process.
* There are four major changes done across stages, including:
    * Some words have changed, like for example we want the sheep riding on the cart to be mysterious until when the player sees it, by changing "a sheep is chasing you" to "someone is ahead of you".
    * On stage 6 (the last complete so far), when player lost to the sheep, it can now tell a player to try it again, and teleport back to start (until the play wins over the sheep).
    * Added resetting feature throughout the stages (on stages 2 and 5 it will clear inventory, and specifically to stage 5 the holes and webs will be reverted back), it wasn't exist during it's true development.
    * On stage 3, filling procedures has been minorly adjusted. It will now fill the blocks with planks and then place the goal (green wool) to where it supossed. This did not exist before.
