# Dual-mode pump
This is the Oitswilliam-designed multi-purpose pump. It can be used as vacuum bag pump or air pump, or both. Please check its CAD document on how to assemble it yourself.

## BOM
* 1X 5V 370 motor with air pump
* 1X DS-211 latching push-button switch
* 1X USB-C device port

## Printable files legend
* [flex] - require TPU flexible filament, Shore hardness doesn't matter as long as it's at least 95A.
